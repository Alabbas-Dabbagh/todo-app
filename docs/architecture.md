# 1. Architectural Overview

The application follows a simple component-based architecture using React Native.

The system consists of three main layers:

1. User Interface (UI Layer)
2. State Management Layer
3. Persistence Layer (AsyncStorage)

---

# 2. Component Description

## UI Layer
- TextInput for entering tasks
- Add Button to create new tasks
- FlatList to display tasks
- Delete Button for removing tas
